好き嫌い.com 非公式ブラウザ

好きか、嫌いか。
もっと手軽に。

好き嫌い.com を快適に楽しめる非公式アプリ。 ランキング・投票・コメント・ブックマークをスマートフォンで。

Features

全部入りで、使いやすい

ランキング
好感度ランキングをチェック
好感度・不人気・トレンドの3タブで最新ランキングを確認。無限スクロールで次々閲覧できます。
スワイプ投票
スワイプで快速投票
カードを右にスワイプで「好き」、左で「嫌い」。未投票の人物をテンポよく評価できます。
詳細・コメント
コメントを読んで投票
好き派・嫌い派のコメントをフィルタ表示。コメントへのgood/bad投票や返信投稿も可能。
スレ内検索
コメントをキーワード検索
気になるキーワードでスレッド内を絞り込み。大量のコメントから目的の発言をすぐに見つけられます。
ブックマーク
フォルダで整理・管理
「俳優」「アイドル」など自由にフォルダを作成。気になる人物をカテゴリ別にブックマーク管理できます。
履歴
投票・閲覧・コメント履歴
過去の投票・閲覧・コメント投稿が時系列で確認できます。あの人物の詳細にすぐ戻れます。
